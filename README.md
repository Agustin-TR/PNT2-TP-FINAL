## Instructions
### Spanish (Original)
Realizar una aplicación frontend basada en Vue CLI, con tema de elección libre,
que integre los temas que se detallan a continuación:
• Vue CLI 3
• Framework CSS (Bootstrap, Materialize, Material)
• Componentes (archivo único, múltiples archivos)
• Uso de métodos y propiedades computadas
• Directivas Data Binding (v-bind ‘:’, v-on ‘@’, v-model)
• Directivas estructurales (v-if, v-show, v-for, v-else, v-else-if)
• Directivas de atributos (:class :style)
• Lifecycle Hooks
• Manejo de Props y eventos
• Formulario con validaciones
• Routeo de componentes con parámetros (manejados desde HTML y desde código)
• Implementación de una API Rest Full mediante axios (con async/await)
• Uso de patrón de estado global

Se podrá utilizar el backend desarrollado en otra materia (Ej, el backend Node.js
de TP2). Sólo se calificará lo realizado en el frontend.
El trabajo se realizará en equipo (de 3 a 4 integrantes) y se presentará en forma
grupal para su defensa en la última fecha de cursada.

### English (translation)
Develop a frontend application based on Vue CLI, with a freely chosen theme,
that integrates the following topics:
• Vue CLI 3
• CSS Framework (Bootstrap, Materialize, Material)
• Components (single-file, multi-file)
• Use of methods and computed properties
• Data Binding Directives (v-bind ‘:’, v-on ‘@’, v-model)
• Structural Directives (v-if, v-show, v-for, v-else, v-else-if)
• Attribute Directives (:class :style)
• Lifecycle Hooks
• Handling Props and events
• Form with validations
• Component routing with parameters (handled from HTML and from code)
• Implementation of a RESTful API using axios (with async/await)
• Use of a global state pattern

You may use the backend developed in another course (e.g., the Node.js backend from TP2).
Only the frontend work will be graded.
The project will be carried out in teams (3 to 4 members) and presented as a group for defense on the final class date.

## Dependencies
To install all the project dependencies, please run the following commands on your terminal

**Clone the Repository**
`git clone https://github.com/Agustin-TR/PNT2-TP-FINAL.git`

**Install the dependencies**
`npm init -y && npm i axios@^1.12.2 bootstrap@^5.3.8 bootstrap-icons@^1.13.1 dotenv@^17.2.3 lodash.debounce@^4.0.8 pinia@^3.0.4 vue@^3.5.22 vue-router@^4.5.1`
